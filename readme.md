🤖🎨 AI Holi Celebration

An interactive Holi-themed web experience where users can celebrate Holi with AI bots by spraying colorful Holi water.

This project combines festival creativity with AI imagination to create a fun browser-based experience.

🚀 Features

✨ Cinematic futuristic background
🤖 Multiple AI bots standing on screen
🎨 Color pots to choose Holi colors
🔫 Cursor changes into spray mode
💦 Click anywhere to spray colors on AI bots
🎉 Interactive and fun Holi experience

🖼 Concept

The idea behind this project:

What if AI bots celebrated Holi?

Users can select colors and spray them on AI bots to create a colorful AI Holi celebration.

Each color represents the diversity of AI technologies, similar to how Holi celebrates diversity through colors.
